# **App Name**: ExcelView

## Core Features:

- Excel Import: Import an Excel file (xls or xlsx format).
- Column Limitation: Restrict imported columns to A through E.
- Table Display: Display the imported data in a table format.
- Pagination: Implement pagination for large datasets.

## Style Guidelines:

- Primary color: Light blue (#A7D1F2), reminiscent of spreadsheet software but with a modern feel.
- Background color: Very light gray (#F0F0F0), to provide a clean backdrop.
- Accent color: Soft green (#B5E487), used for interactive elements like pagination buttons and import confirmations.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, creating a clean and readable interface.
- Simple, professional icons for file import and table navigation.
- Clean and structured layout, with the table taking center stage. Clear separation of controls (import button, pagination) from the data display.
- Subtle transitions for pagination and table updates.
